# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-09

### Added
- Initial public release
- Voice control with offline STT/TTS
- Offline AI inference (llama.cpp)
- Device automation (20+ commands)
- Cloud synchronization with encryption
- Multi-language support (14 languages)
- Voice biometric authentication
- Image generation via Replicate API
- Web search integration
- Advanced security features
- Material Design 3 UI
- Jetpack Compose implementation
- Hilt dependency injection
- Comprehensive documentation

### Security
- AES-256-GCM encryption for local data
- Android Keystore integration
- OAuth 2.0 authentication
- JWT token management
- Input validation and output encoding
- Rate limiting

## [1.0.0] - 2025-12-01

### Initial Development
- Core architecture
- Basic voice features
- Device control framework
- Database setup
